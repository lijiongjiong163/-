# dict(无序的键值对)

### 1.用{}来创建

```python
>>> d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}
>>> d['Michael']
95
```

也可以单独赋值

```python
>>> d['Adam'] = 67
>>> d['Adam']
67
```

### 2.dict是用空间来换取时间的一种方法，占空间大，查找极快。

# set(无序不可重复)

### 1.要创建一个set，需要提供一个list作为输入集合：

```python
>>> s = set([1, 2, 3])
>>> s
{1, 2, 3}
```

## 1.基本用法

`Promise`对象是一个构造函数，用来生成`Promise`实例。

```javascript
var a =parseInt(100*Math.random());//生成一个100内随机整数，用于模拟成功|失败
//1.new一个promise实例，要传一个函数作为参数，这个函数有两个形参，这两个形参也都是传入函数。名字随便起，但是约定好是resolve和reject，系统自动往进传
const p = new Promise(function(resolve, reject){
  console.log('正在获取后台数据。。。');
  setTimeout(()=>{
    if(a%2==0){
     //2.1调用此函数，会将Promise对象的状态从执行中变成成功
    resolve('后台获取数据成功啦！！！');
 }else{
     //2.2调用此函数，会将Promise对象的状态从执行中变成失败
   reject('报错！！')
 }},1000)
    
});
//3.再用这个promise实例的then方法去调用它，调用时需要绑定回调函数，then需要传两个参数，都是函数，第一个是成功的回调，第二个是失败的回调，这两个方法接收上面resolve和reject的返回值。
p.then(function(value){
  console.log(value);
},function(reason){
  console.log(reason);
})

```

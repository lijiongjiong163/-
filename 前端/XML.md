## 1.XML包含

#### 1.文档声明

```xml
<?xml version="1.0" encoding="ISO-8859-1"?>
```

version：xml版本

encoding：此文件的编码

#### 2.语法规则

1. 元素名称不能带空格

   ```xml
   <bo ok>书</bo ok>
   ```

2. 大小写敏感（开始标签和结束标签必须一致大小写，html则可不用）

3. xml文档必须有根元素，就是必须有一个顶级元素，吧其它元素都包含着，不能说是一个文件里同时存在着两个根元素

4. 特殊字符：<>要想在元素中间的字符中用这两个字符，请用\&lt;(<)和\&gt;(>)

#### 3.文本区域（CDATA区域）

CDATA区域中只是纯文本，不会进行xml语法解析。

格式如下：

<![CDATA[
	<不会被解析>
]]>

```xml
<![CDATA[文本内容]]>
```
